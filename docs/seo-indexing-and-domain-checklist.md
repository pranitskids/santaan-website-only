# Santaan SEO Indexing And Domain Checklist

This site should use `https://www.santaan.in` as the canonical public website.

## Search Console

1. Submit `https://www.santaan.in/sitemap.xml` in Google Search Console.
2. Inspect these URLs and request indexing after deployment:
   - `https://www.santaan.in/ivf-cost-in-india-2026`
   - `https://www.santaan.in/ivf-cost-bhubaneswar`
   - `https://www.santaan.in/ivf-cost-berhampur`
   - `https://www.santaan.in/ivf-cost-bangalore`
   - `https://www.santaan.in/iui-cost`
   - `https://www.santaan.in/icsi-cost`
   - `https://www.santaan.in/egg-freezing-cost`
   - `https://www.santaan.in/founder-notes`
3. Also inspect representative archive URLs after Medium sync:
   - `https://www.santaan.in/fertility-insights`
   - `https://www.santaan.in/clinical-insights`
   - One recent patient article URL
   - One recent doctor article URL
4. If Google reports "Discovered, not indexed", wait after the first crawl and add internal links from relevant Medium posts or Santaan pages.
5. If Google chooses a different canonical, check whether an old domain or Medium URL is competing with the Santaan URL.

## Domain Consolidation

1. Keep `www.santaan.in` as the canonical domain in page metadata and sitemap.
2. Keep `santaan.in` redirecting to `www.santaan.in`.
3. If `santaanivf.in` is revived, redirect it permanently to `https://www.santaan.in`.
4. If `santaanmedicare.com` is a separate business, keep it clearly separate. If it contains Santaan IVF/fertility pages, redirect or canonicalize those pages to `www.santaan.in`.

## Publishing Rule

For founder or investor content, publish on Santaan first, then syndicate to LinkedIn or Medium with a canonical link back to Santaan. This keeps authority on the main domain.
