# CEO Growth Manual (Presentation Deck)

Use this as a PPT outline. Each `---` is one slide.

---

## 01. CEO Growth Manual
**Goal:** predictable growth with measurable ROI (no external reporting)

**CRM is the single source of truth**
- Campaign spend + notes → Ops Inputs / Spend
- Lead handling + closure → Contacts / Follow-ups
- CEO sees truth → CEO Command Center

---

## 02. The Growth Rule
**No WhatsApp reports / spreadsheets.**

Everything happens inside CRM:
- Teams input their responsibilities in CRM
- CEO dashboard aggregates and shows actionable info

What CEO checks daily:
- Wiring Health (NeoDove/calls/WhatsApp/spend last seen)
- NeoDove 7-day trend
- CPL / CPP / conversion rate
- SLA breaches + pending >24h + overdue follow-ups

---

## 03. CEO Daily Cadence (My Day)
**09:30 AM** — CEO Command: wiring + ROI + leaks → assign owners and deadlines  
**11:00 AM** — Agency spend/metrics must be updated (Ops Inputs → Agency)  
**03:30 PM** — Closure huddle: conversions + blockers + script fixes  
**09:00 PM** — Close day: Daily Command + Ops Workboard → compliance and closure note  

---

## 04. Growth Scoreboard (Simple)
Demand:
- leads per center/day
- NeoDove calls/day
- high-intent leads (score ≥ 70)

Closure:
- converted patients
- qualified → converted closure
- lost rate + winback

Efficiency:
- CPL, CPP
- attribution coverage ≥ 85%

---

## 05. TV + Billboard Planning (Like Performance)
Before launch:
- unique QR/keyword per creative = asset ID
- landing path includes center
- telecaller script matches the ad promise
- define daily spend cap + minimum conversion target

During run:
- TV spots logged daily (Ops Inputs → TV Ads)
- CEO checks next morning: call volume + conversions
- change creative if no lift after 3 days

---

## 06. Digital Campaigns (Meta/Google/Social)
**No UTM = no spend.**

CEO checklist:
- message match: ad → landing → call script
- UTM template for every link
- spend logged daily by 11:00 AM
- CEO Command shows CPL/CPP by campaign

---

## 07. Agency Management System
Weekly CEO review:
- top 3 campaigns by conversions + CPP
- bottom 3 campaigns by conversion rate
- attribution gaps (unattributed leads)
- creative refresh + landing changes plan

Hard rules:
- spend logged daily in CRM
- any campaign without UTMs paused
- underperformers flagged by 03:00 PM with corrective note

---

## 08. Telecaller Unit Management
The biggest lever is speed-to-lead + follow-up discipline.

CEO non-negotiables:
- 2-hour SLA breaches → must trend to zero
- pending >24h → must trend to zero
- every lead has: Status + Owner + Next follow-up time

Daily sync cycles:
- 11:00 AM: Cycle A (hot leads)
- 03:00 PM: Cycle B (callbacks)
- 07:00 PM: Cycle C (reconciliation)

---

## 09. IT Team Management (Reliability)
IT owns:
- wiring health stays green
- attribution coverage stays above 85%
- NeoDove webhook updates owner + follow-up fields
- CEO dashboard loads without errors

CEO escalation triggers:
- NeoDove/calls last seen > 2h during business hours
- attribution < 75%
- spend present but conversions flat (possible wiring/closure mismatch)

---

## 10. Weekly Operating Rhythm (Meetings)
Growth review (45m): scale winners, pause losers  
Closure review (30m): scripts + slots + counselling fixes  
Ops discipline (20m): compliance gaps + owners  
IT reliability (15m): wiring + attribution + uptime  

---

## 11. 30-Day Plan
Week 1: adoption + wiring green + spend logging  
Week 2: fix attribution + SLA breaches + closure scripts  
Week 3–4: scale winners by CPP + TV/OOH asset tracking + winback  

Month-1 rule:
- Every lead has Owner + Next follow-up time

---

## 12. CEO Decision Rules (7)
Scale:
- only stable CPP + rising conversions
- budget increases 10–15% steps

Stop-loss:
- spend up + conversions flat → pause + fix closure
- attribution down → pause until UTMs fixed
- SLA breaches up → manpower to calls before more ads
- follow-ups overdue → enforce follow-up timestamps
