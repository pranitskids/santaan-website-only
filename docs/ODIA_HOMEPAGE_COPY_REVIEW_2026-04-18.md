# Odia Homepage Copy Review

Date: 2026-04-18
Status: Review only
Instruction: Do not publish yet. This is for language and tone approval first.

## Review goal

Create a first-pass Odia layer for the homepage that feels:

- private
- respectful
- simple
- reassuring
- local without sounding childish

This draft is intentionally short.
It should be reviewed by:

- one native Odia speaker from the content team
- one clinic/counseling lead
- one marketing approver

## Section 1: Hero headline options

### Option A

English reference:
`When waiting has started to feel too long, we're here to talk.`

Odia draft:
`ଅପେକ୍ଷା ବହୁତ ଦିନ ହେଇଗଲା ବୋଲି ଲାଗୁଛି? ଆସନ୍ତୁ, ଶାନ୍ତିରେ କଥା ହେବା।`

English reference:
`If "not yet" has lasted too long, Santaan is here with answers and care.`

`"ଏପର୍ଯ୍ୟନ୍ତ ହେଉନି" ବହୁତ ଦିନ ହେଲା କି? ଉତ୍ତର ଓ ସହଯୋଗ ପାଇଁ ସନ୍ତାନ ଆପଣଙ୍କ ସହିତ ଅଛି।`

### Option C

English reference:
`Private, evidence-based fertility care for couples who want clarity without pressure.`

Odia draft:
`ଚାପ ଛଡ଼ା ସ୍ପଷ୍ଟ ଉତ୍ତର ଚାହୁଁଥିବା ଦମ୍ପତିଙ୍କ ପାଇଁ ନିଜସ୍ୱ, ବିଜ୍ଞାନଭିତ୍ତିକ ଫର୍ଟିଲିଟି କେୟାର।`

## Section 6: Short FAQ review

### FAQ 1

English:
`Do we need IVF immediately?`

English:
`Can we start privately?`

Odia draft:
`ଆମେ ଗୋପନୀୟ ଭାବରେ ଆରମ୍ଭ କରିପାରିବୁ କି?`

Answer draft:
`ହଁ। ଆପଣ WhatsApp ରୁ ଆରମ୍ଭ କରିପାରିବେ ଏବଂ ଆଗକୁ କିପରି ବଢିବେ ସେଥିରେ ଆମେ ଗାଇଡ୍ କରିବୁ।`

Answer draft:
`ହଁ। ଆମ ଉଦ୍ଦେଶ୍ୟ ହେଲା ଆପଣଙ୍କୁ ଚାପ ଦେବା ନୁହେଁ, ସ୍ପଷ୍ଟ ଭାବରେ ବୁଝାଇଦେବା।`
